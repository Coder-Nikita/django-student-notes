# Django Student Notes App

A beginner CRUD project built using Django.

## Features

- Add Notes
- View Notes
- Update Notes
- Delete Notes
- Bootstrap UI

## Tech Stack

- Python
- Django
- Bootstrap

## Installation

Clone the repository:

git clone https://github.com/Coder-Nikita/django-student-notes.git

Go to project folder:

cd django-student-notes

Install dependencies:

pip install -r requirements.txt

Run server:

python manage.py runserver
